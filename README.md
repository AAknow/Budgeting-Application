# Budgeting-Application
A web based budgeting application

Server Startup:

1. Navigate to "~/repos/Budgeting-Application/server" in the terminal
2. Run command "node index.js"
3. The port should now be open

Client Startup:

1. Navigate to "~/repos/Budgeting-Application/client" in the terminal
2. Run command "npm start"
3. The web page should appear in the browser
